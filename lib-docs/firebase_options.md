# firebase_options

> **Source:** `lib/firebase_options.dart`

## Purpose

Auto-generated Firebase configuration file produced by the FlutterFire CLI. Provides platform-specific `FirebaseOptions` for initializing Firebase services (Auth, Firestore, Analytics, Performance, Crashlytics). Should not be edited manually — re-run `flutterfire configure` to regenerate.

## Dependencies

- `package:firebase_core/firebase_core.dart` — `FirebaseOptions` type
- `package:flutter/foundation.dart` — `defaultTargetPlatform`, `kIsWeb`, `TargetPlatform`

## Notes

- Generated by FlutterFire CLI — do not edit manually.
- iOS and macOS share the same `appId`, `apiKey`, `iosClientId`, and `iosBundleId`.
- Web and Windows share the same `appId` and `apiKey`.
- Supported platforms: Web, Android, iOS, macOS, Windows. Linux is not configured.
